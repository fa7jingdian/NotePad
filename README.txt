# 记事本应用 (NotePad)

## 项目概述
这是一个功能完善的Android记事本应用，提供了笔记管理、分类管理、搜索等核心功能，支持中文界面和中文内容操作。

## 主要功能

### 1. 笔记管理
- **创建笔记**：支持创建新笔记，设置标题和内容
- **编辑笔记**：修改现有笔记的标题和内容
- **删除笔记**：单个删除或批量删除笔记
- **查看笔记**：以列表形式查看所有笔记，按分类和时间排序

### 2. 分类管理
- **创建分类**：添加新的笔记分类
- **编辑分类**：修改现有分类名称
- **删除分类**：删除不再需要的分类
- **分类显示**：笔记按分类分组显示

### 3. 搜索功能
- **全文搜索**：支持对笔记标题、内容和分类进行搜索
- **中文搜索**：完全支持中文关键词搜索
- **实时搜索**：搜索结果实时更新

### 4. 主题支持
- **深色主题**：保护眼睛，适合夜间使用
- **浅色主题**：清晰明亮，适合白天使用
- **主题切换**：通过菜单快速切换主题

### 5. 批量操作
- **批量选择**：长按笔记项进入批量选择模式
- **批量删除**：一次性删除多个选中的笔记

### 6. 数据导出
- **笔记导出**：支持将笔记导出为文本文件
- **数据备份**：可备份笔记内容到设备存储

## 技术栈

- **开发语言**：Java
- **开发环境**：Android Studio
- **目标SDK**：Android API 30+
- **数据库**：SQLite (Android内置)
- **UI框架**：Android原生UI组件
- **架构模式**：MVC (Model-View-Controller)

## 安装方法

### 1. 使用Android Studio安装
1. 克隆或下载项目
2. 使用Android Studio打开项目
3. 连接Android设备或启动模拟器
4. 点击"Run"按钮安装应用

### 2. 手动安装APK
1. 在项目根目录执行：`./gradlew assembleDebug`
2. 从`app/build/outputs/apk/debug/`目录获取APK文件
3. 将APK文件复制到Android设备
4. 在设备上安装APK文件

## 使用说明

### 创建新笔记
1. 在主界面点击右上角的"新建"按钮
2. 输入笔记标题和内容
3. 选择分类（可选）
4. 点击"保存"按钮

### 编辑笔记
1. 在笔记列表中点击要编辑的笔记
2. 修改标题或内容
3. 点击"保存"按钮

### 分类管理
1. 在主界面点击菜单中的"分类管理"
2. 点击"添加分类"创建新分类
3. 长按分类项可编辑或删除分类

### 搜索笔记
1. 在主界面点击搜索图标
2. 输入搜索关键词
3. 搜索结果会实时显示

### 主题切换
1. 在主界面或编辑界面点击菜单
2. 选择"切换主题"选项
3. 应用会立即切换到另一种主题

### 批量删除
1. 在主界面长按任意笔记项进入批量选择模式
2. 勾选要删除的笔记
3. 点击"删除"按钮确认删除

## 中文支持

应用完全支持中文：
- 界面元素全部使用中文显示
- 支持中文笔记内容的创建和编辑
- 支持中文分类名称
- 支持中文关键词搜索

## 开发环境配置

### 1. 必要工具
- Android Studio Arctic Fox (2020.3.1) 或更高版本
- Java JDK 8 或更高版本
- Android SDK API Level 30 或更高版本

### 2. 项目配置
- 打开项目后，Android Studio会自动下载所需依赖
- 确保已安装必要的Android SDK版本
- 配置好Android模拟器或连接真实设备

## 数据存储

应用使用SQLite数据库存储数据，主要表结构：

### notes表
- `_id`：笔记ID（主键）
- `title`：笔记标题
- `note`：笔记内容
- `created`：创建时间
- `modified`：修改时间
- `category_id`：分类ID（外键）

### categories表
- `_id`：分类ID（主键）
- `title`：分类名称

## 权限说明

应用需要以下权限：
- `WRITE_EXTERNAL_STORAGE`：用于导出笔记到外部存储
- `READ_EXTERNAL_STORAGE`：用于读取外部存储的文件
- `INTERNET`：用于可能的在线功能（当前未使用）
