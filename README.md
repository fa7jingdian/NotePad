# NotePad 记事本应用

这是一个基于Android官方示例扩展开发的记事本应用，提供了笔记的创建、编辑、删除、搜索等基础功能，并增加了**分类管理**和**笔记导出**等实用功能。

## 功能特点

### 基础功能
- 创建、编辑和删除笔记
- 笔记列表展示，支持按修改时间排序
- 搜索笔记（按标题和内容）
- 复制和粘贴笔记内容

### 新增功能

#### 1. 分类管理
- 支持创建多个笔记分类
- 新建笔记时可选择所属分类
- 分类管理界面，可添加、编辑和删除分类
- 笔记默认归类到"默认分类"

#### 2. 笔记导出
- 导出全部笔记到文本文件
- 按分类导出笔记
- 单个笔记导出
- 导出文件保存在设备的`文档/NotePad`目录下
- 导出文件包含笔记标题、内容、创建时间和修改时间

#### 3. 主题切换
- 支持明暗主题切换
- 适配系统主题设置

## 使用说明

### 分类功能使用
1. 在主界面点击菜单中的"分类管理"进入分类管理界面
2. 点击菜单可添加新分类
3. 长按现有分类可编辑或删除
4. 在创建或编辑笔记时，可通过下拉菜单选择分类

### 导出功能使用
1. 在主界面点击菜单中的"导出笔记"
2. 选择"导出全部"导出所有笔记
3. 选择"按分类导出"后，在弹出的分类列表中选择要导出的分类
4. 在编辑笔记界面，可通过菜单直接导出当前笔记

## 技术实现

- **数据存储**：使用SQLite数据库存储笔记和分类信息
- **内容提供者**：通过ContentProvider实现数据访问抽象
- **列表展示**：使用ListView和SimpleCursorAdapter展示数据
- **主题适配**：实现了支持明暗主题的UI适配

## 项目结构

```

├── NoteEditor.java       # 笔记编辑界面
├── NotesList.java        # 笔记列表主界面
├── CategoryListActivity.java  # 分类管理界面
├── ExportManager.java    # 导出功能管理器
├── ThemeManager.java     # 主题管理类
└── NotePad.java          # 常量和URI定义
```

## 开发环境

- Android SDK
- Android Studio
- Java

## 注意事项

- 导出功能需要存储权限
- 删除分类时，如果分类中包含笔记，将无法删除
- 导出的文本文件采用UTF-8编码
